# FeelUP – Mood Tracking App

FeelUP is a simple mood tracking web application that allows users to record their daily emotional state and review recent mood history.

The app stores data locally in the browser and calculates a weekly average mood score.

## Live Demo

https://leogos.github.io/feelup/

## Features

- Add daily mood entries
- View recent mood history
- Automatic weekly average calculation
- Data persistence using LocalStorage
- Dark mode toggle support
- Clean responsive UI

## Technologies

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage API

## Project Purpose

This project was developed to practice JavaScript-based UI interaction, event handling, and browser storage management in a small standalone frontend application.
